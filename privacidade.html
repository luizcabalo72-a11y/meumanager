Política de Privacidade

Última atualização: Janeiro de 2026

Resumo: Respeitamos sua privacidade. Coletamos apenas dados necessários para fornecer o serviço. Não vendemos seus dados. Você tem controle total sobre suas informações.

1. Quem Somos

O Meu Manager é um serviço de gestão de vendas desenvolvido no Brasil. Esta política explica como coletamos, usamos e protegemos seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

2. Dados que Coletamos

Dados de cadastro:

  • Nome
  • Email
  • Nome da empresa
  • Telefone (opcional)

Dados de uso:

  • Produtos, compras e vendas cadastrados
  • Configurações do sistema
  • Logs de acesso (data, hora, IP)

Dados de pagamento:

  • Processados diretamente pelo Stripe
  • Não armazenamos dados de cartão de crédito

3. Como Usamos seus Dados

  • Fornecer e melhorar o serviço
  • Processar pagamentos
  • Enviar comunicações sobre o serviço
  • Suporte ao cliente
  • Cumprir obrigações legais

4. Base Legal (LGPD)

Tratamos seus dados com base em:

  • Execução de contrato: para fornecer o serviço
  • Consentimento: para comunicações de marketing
  • Legítimo interesse: para melhorias e segurança
  • Obrigação legal: para fins fiscais e regulatórios

5. Compartilhamento de Dados

Compartilhamos dados apenas com:

  • Firebase (Google): armazenamento e autenticação
  • Stripe: processamento de pagamentos
  • Autoridades: quando exigido por lei

Não vendemos nem alugamos seus dados para terceiros.

6. Seus Direitos

De acordo com a LGPD, você tem direito a:

  • Acesso: saber quais dados temos sobre você
  • Correção: corrigir dados incorretos
  • Exclusão: solicitar a eliminação dos dados
  • Portabilidade: exportar seus dados
  • Revogação: retirar consentimento a qualquer momento

Para exercer esses direitos, envie email para: privacidade@meumanager.com.br

7. Segurança

Implementamos medidas de segurança incluindo:

  • Criptografia em trânsito (HTTPS/TLS)
  • Criptografia em repouso no Firebase
  • Autenticação segura
  • Backups regulares
  • Controle de acesso restrito

8. Cookies

Usamos cookies para:

  • Manter sua sessão ativa
  • Lembrar suas preferências
  • Análise de uso (anônimo)

Você pode configurar seu navegador para bloquear cookies, mas isso pode afetar a funcionalidade do serviço.

9. Retenção de Dados

  • Conta ativa: dados mantidos enquanto usar o serviço
  • Após exclusão: dados removidos em até 30 dias
  • Backups: removidos em até 90 dias
  • Dados fiscais: mantidos por 5 anos (obrigação legal)

10. Menores

O serviço não é destinado a menores de 18 anos. Não coletamos dados de menores intencionalmente.

11. Alterações

Podemos atualizar esta política periodicamente. Alterações significativas serão notificadas por email.
